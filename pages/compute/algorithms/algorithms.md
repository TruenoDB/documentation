## Compute Menu
1. Compute Tab.
2. Algorithms type.
3. Algorithm name.
4. Algorithm's properties (e.g. **R=Remote** [Computed on Spark-GraphX], **D=Directed**, **UD=Undirected**, **UW=Unweighted**, **W=Weighted**)
5. Algorithm's description.
